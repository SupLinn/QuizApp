Tech Stack:- <br />
-> Frontend: React.js, Tailwind CSS, Vite <br />
-> Backend: Node.js, Express.js <br />
-> Database: MongoDB (Mongoose) <br />
-> Deployment: Vercel (Frontend) + Render (Backend) <br />

Features Completed:- <br />
-> Topic Selection: Dynamic homepage displaying available quiz categories (JavaScript, React, Node.js, MongoDB). <br />
-> Interactive Quiz Interface: Displays one question at a time with instant state management. <br />
-> Result Summary: Visual breakdown of Total Score, Correct answers, and Incorrect attempts. <br />
-> Detailed Review Page (Bonus): comprehensive report card showing user answers vs. correct answers for every question. <br />
-> Responsive UI: Fully mobile-optimized layout with a clean, modern design. <br />
