Tech Stack:-
-> Frontend: React.js, Tailwind CSS, Vite
-> Backend: Node.js, Express.js
-> Database: MongoDB (Mongoose)
-> Deployment: Vercel (Frontend) + Render (Backend)

Features Completed:-
-> Topic Selection: Dynamic homepage displaying available quiz categories (JavaScript, React, Node.js, MongoDB).
-> Interactive Quiz Interface: Displays one question at a time with instant state management.
-> Result Summary: Visual breakdown of Total Score, Correct answers, and Incorrect attempts.
-> Detailed Review Page (Bonus): comprehensive report card showing user answers vs. correct answers for every question.
-> Responsive UI: Fully mobile-optimized layout with a clean, modern design.

Future Improvements:-
-> Leaderboard: Global ranking system to compare scores with other users.
-> User Authentication: Login/Sign-up to save history and track progress over time.
-> Progress Bar: A visual indicator during the quiz to show completion percentage.
